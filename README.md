# Sea

## Installation

```
$ yarn install
$ yarn build
$ yarn link
```

## Usage

```
Usage: sea [options] [command]

Options:
  -V, --version     output the version number
  -h, --help        output usage information

Commands:
  branch <cmd>      branching
  changes           display changes
  commit <message>  commit changes
  checkout <name>   switch to a branch
  diff              diff changes
  rebase            rebase branch
  help [cmd]        display help for [cmd]
```
